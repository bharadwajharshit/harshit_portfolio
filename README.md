# Professional Digital Portfolio

A vibrant, tech-savvy portfolio website built with modern HTML, CSS (Glassmorphism), and JavaScript (Canvas Particles, Tilt Effects).

## 🚀 How to Deploy

### Option 1: Vercel (Recommended)
1. **Install Vercel CLI**:
   ```bash
   npm i -g vercel
   ```
2. **Deploy**:
   Run the following command in this directory:
   ```bash
   vercel
   ```
   Follow the prompts (hit Enter for defaults). Your site will be live instantly!

### Option 2: GitHub Pages
1. Push this code to a GitHub repository.
2. Go to **Settings > Pages**.
3. Select `main` branch as the source.
4. Your site will be available at `https://<your-username>.github.io/<repo-name>`.

### Option 3: Netlify
1. Drag and drop this folder onto the [Netlify Drop](https://app.netlify.com/drop) page.

## 🛠 Features
- **Dark Mode**: Deep aesthetic with neon accents.
- **Particle Network**: Canvas-based interactive background.
- **Micro-interactions**: 3D Tilt cards, Glitch text, Typewriter effect.
- **Responsive**: Fully optimized for mobile and desktop.
